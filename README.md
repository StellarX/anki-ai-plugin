# Anki AI 助手插件

一个为 Anki 添加 AI 对话功能的插件，通过智谱 AI 的 GLM 模型提供智能问答服务。

## 功能特点

- 🤖 集成智谱 AI GLM 模型，提供智能对话功能
- 💬 支持多轮对话，保持上下文连续性
- 🎨 现代化深色主题界面，美观舒适
- ⌨️ 快捷键支持，快速唤起对话窗口
- 📝 支持 Markdown 格式渲染，代码块高亮
- 🔄 支持新对话功能，随时清空历史

## 安装方法

1. 下载插件文件：`__init__.ankiaddon`
2. 打开 Anki
3. 点击菜单栏的 `工具` > `插件` > `从文件安装插件...`
4. 选择下载的 `__init__.ankiaddon` 文件
5. 重启 Anki
6. 直接访问https://ankiweb.net/shared/info/824809951也可安装

## 配置说明

### 获取 GLM API Key

1. 访问 [智谱 AI 开放平台](https://open.bigmodel.cn/)
2. 注册账号并登录
3. 进入「API Keys」页面创建新密钥
4. 复制密钥到下方 `api_key` 字段

### 配置步骤

1. 打开 Anki
2. 点击菜单栏的 `工具` > `插件`
3. 找到 `anki-plugin` 插件
4. 点击 `配置` 按钮
5. 在配置文件中填写以下内容：

```json
{
  "api_key": "你的API密钥",
  "model": "glm-4-flash",
  "api_url": "https://open.bigmodel.cn/api/paas/v4/chat/completions"
}
```

### 可用模型

- `glm-4-flash` - 轻量级，响应快（推荐）
- `glm-4-air` - 性能更好

## 使用指南

### 快速启动

- **快捷键方式**：在 Anki 中快速按三次 `Ctrl` 键即可打开 AI 对话窗口

### 对话操作

1. 在输入框中输入您的问题
2. 按 `Enter` 键发送消息
3. 按 `Ctrl + Enter` 换行
4. 点击"新对话"按钮可以清空历史记录，开始新的对话

### 界面说明

- **上方区域**：显示 AI 的回复和您的消息历史
- **输入框**：用于输入您的问题
- **发送按钮**：发送消息给 AI
- **新对话按钮**：清空历史记录，开始新对话

## 注意事项

1. 首次使用前必须配置 API Key，否则插件无法正常工作
2. 配置完成后需要重启 Anki 才能生效
3. 确保网络连接正常，插件需要联网才能调用 AI 服务
4. 建议使用 `glm-4-flash` 模型以获得更快的响应速度

## 常见问题

**Q: 插件无法启动？**  
A: 请检查 API Key 是否正确配置，并确保已重启 Anki。

**Q: AI 回复很慢？**  
A: 可以尝试切换到 `glm-4-flash` 模型，它的响应速度更快。

**Q: 如何清空对话历史？**  
A: 点击对话窗口中的"新对话"按钮即可清空历史记录。

**Q: 插件支持哪些 Markdown 格式？**  
A: 支持代码块、表格、引用等常见 Markdown 格式。
